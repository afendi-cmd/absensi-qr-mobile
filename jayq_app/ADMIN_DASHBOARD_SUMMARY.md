## ✨ Design Features

### Typography

- **Modern fonts** dengan proper weight
- **Bold headers** (18px)
- **Large numbers** (28-48px) dengan negative letter spacing
- **Clean labels** (11-13px)

### Spacing

- **Luas & breathable** - tidak cramped
- **Consistent** - 20px margins, 16-28px padding
- **Modern gaps** - 12-16px between elements

### Shadows

- **Soft shadows** - subtle depth
- **Colored shadows** - blue tint pada main card
- **Floating effect** - cards terasa elevated

### Corners

- **Large rounded corners** - 24-28px
- **Medium rounded** - 16-20px untuk icons
- **Small rounded** - 12-14px untuk badges

### Icons

- **Rounded style** - modern icon set
- **Colored backgrounds** - dengan alpha 0.1
- **Proper sizing** - 24-56px

## 🔧 Customization

### Ganti Warna Primary

```dart
// Cari dan ganti:
Color(0xFF2563EB) // dengan warna pilihan Anda
```

### Ubah Data

Edit data di dalam widget:

- `_buildMainStatCard()` - Mata kuliah aktif
- `_buildStatsGrid()` - Total dosen & mahasiswa
- `_buildJadwalList()` - Jadwal perkuliahan

### Tambah Menu

Edit `_buildAksiCepat()` untuk menambah quick action menu.
